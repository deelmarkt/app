# DeelMarkt Branding Assets

## Required Assets

### `/logo/`
- `deelmarkt-logo-full.svg` — full logo with windmill + text (horizontal)
- `deelmarkt-logo-full.png` — PNG export (2x, 3x)
- `deelmarkt-wordmark.svg` — "deelmarkt" text only

### `/icons/`
Individual icon elements extracted from the logo:
- `icon-windmill.svg`
- `icon-bicycle.svg`
- `icon-car.svg`
- `icon-box.svg` (box with teddy bear + camera)
- `icon-location-pin.svg`

### `/app-icon/`
- `app-icon-1024.png` — master (1024x1024, no transparency for iOS)
- Exported via Flutter's asset pipeline to all required sizes

## Brand Colours (from logo)

| Colour | Hex | Usage |
|:-------|:----|:------|
| Deep Navy | `#1a2e4a` | "markt" text, primary dark |
| Warm Orange | `#e85d24` | "deel" text, CTAs, accent |
| Green | `#4caf50` | Location pin, trust/verification |
| Sky Blue | `#87ceeb` | Background accents |
